# Proyecto: Plantilla de Experiencia de Usuario Interactiva

## Descripción

Este proyecto es una plantilla interactiva para gestionar tablas de experiencia de usuario. Permite crear, editar y guardar tablas dinámicas que representan diferentes etapas, puntos de contacto y niveles de experiencia del cliente. Además, incluye funcionalidades para exportar las tablas a PDF y sincronizarlas con Firebase.

## Tecnologías Utilizadas

- **HTML5**: Estructura del contenido.
- **CSS3**: Estilización y diseño responsivo.
- **JavaScript**: Lógica de la aplicación y manipulación del DOM.
- **Firebase**: Almacenamiento y sincronización de datos.
- **jsPDF**: Generación de archivos PDF.

## Estructura del Proyecto

- **index.html**: Contiene la estructura principal de la página, incluyendo los elementos interactivos como botones y modales.
- **style.css**: Define los estilos visuales de la página, incluyendo diseño responsivo y personalización de elementos.
- **script.js**: Implementa la lógica de la aplicación, como la gestión de tablas, la interacción con Firebase y la exportación a PDF.

## Funcionalidades

1. **Crear Tablas**: Permite crear nuevas tablas dinámicas.
2. **Editar Tablas**: Habilita la edición de celdas y títulos.
3. **Guardar Tablas**: Sincroniza las tablas con Firebase.
4. **Exportar a PDF**: Genera un archivo PDF de la tabla actual.
5. **Gestión de Tablas Guardadas**: Visualiza, renombra y elimina tablas almacenadas en Firebase.

## Instalación y Uso

1. Clona este repositorio.
2. Abre el archivo `index.html` en un navegador.
3. Configura las credenciales de Firebase en `script.js`.
4. ¡Comienza a interactuar con las tablas!
